# Sistem Informasi Perikanan Tangkap
## Dinas Perikanan Kabupaten Gresik

Aplikasi web untuk manajemen data BBM, file pengajuan, dan SOP Perikanan Tangkap.

### Fitur
- **Admin Panel**: CRUD Data BBM, Upload File, Manage SOP
- **User Panel**: Lihat Data BBM, Download File, Lihat SOP

### Teknologi
- PHP Native
- MySQL
- Bootstrap 5
